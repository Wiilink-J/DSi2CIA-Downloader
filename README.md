# DSi2CIA-Downloader
This is a program that let you install DSi System apps cias to then use them on 3DS, including the DSi System Settings. The file is an .exe file and is advaible in different languages: English, French, Spanish, Italian, Netherlands, German, Portuguese (included Brasilian portuguese), Russian and Japanese
The tool will install on a cia folder the cias and the files to make DSi System Settings work with a .txt file as instructions.

In the .zip file there are the program but also another .zip files with the files of DSi System Settings

hope you enjoy the project
