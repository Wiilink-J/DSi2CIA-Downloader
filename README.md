# DSi2CIA Downloader

This is a program that lets you install DSi System app CIAs to use them on a 3DS, including the DSi System Settings. 

### 🌍 Supported Languages
The tool executable (.exe) is available in multiple languages:
* English
* French
* Spanish
* Italian
* Netherlands
* German
* Portuguese (including Brazilian Portuguese)
* Russian
* Japanese

### 📦 Package Contents
* Main program (.exe and .py versions)
* A `.zip` archive containing the necessary DSi System Settings files

---

## 🛠️ Installation & Requirements

> [!IMPORTANT]
> **Windows x64 Users (Windows 10/11):** You can directly run the GUI `.exe` file included in the zip.
> **Other Operating Systems:** If you are running 32-bit Windows, macOS, or Linux, ensure you have Python installed and follow the specific instructions below to run the `.py` version in the zip file.

---

### Windows (32-bit / x32)

1. Download and install Python from the official website. 
   *(Make sure to check the box **"Add python.exe to PATH"** during installation).*
2. Open the **Command Prompt** (search for `cmd` in the Windows start menu).
3. Install the required library by running:
   ```bash
   pip install customtkinter
   ```
4. To run the app, double-click the `DSi2CIA Downloader.py` file or type this in the Command Prompt:
   ```bash
   python "DSi2CIA Downloader.py"
   ```

---

### 🍎 macOS (Mac)

1. Open the **Terminal** (press `Cmd + Space`, type `Terminal`, and press `Enter`).
2. Install the required library by running:
   ```bash
   pip3 install customtkinter
   ```
   *(Note: If your system says `pip3` or `python3` is missing, download and run the installer from the official Python website first).*
3. To run the app, type this in the Terminal:
   ```bash
   python3 "DSi2CIA Downloader.py"
   ```

---

### 🐧 Linux (Ubuntu, Debian, Mint, etc.)

Linux systems usually have Python pre-installed, but they often lack the package manager (`pip`) and graphical user interface support (`tkinter`) by default.

1. Open the **Terminal** (shortcut: `Ctrl + Alt + T`).
2. Install the required system components by running:
   ```bash
   sudo apt update && sudo apt install python3-pip python3-tk -y
   ```
3. Install the required library by running:
   ```bash
   pip3 install customtkinter
   ```
4. To run the app, type this in the Terminal:
   ```bash
   python3 "DSi2CIA Downloader.py"
   ```
